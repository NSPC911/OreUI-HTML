There are only 6 colors currently available and you can set it through the oreui-color attribute

Light Button

<div oreui-type="button" oreui-color="light">Light Button</div>

Red Button

<div oreui-type="button" oreui-color="red">Red Button</div>

Green Button

<div oreui-type="button" oreui-color="green">Green Button</div>

Purple Button

<div oreui-type="button" oreui-color="purple">Purple Button</div>

Pink Button

<div oreui-type="button" oreui-color="pink">Pink Button</div>

You can also change the active color by setting the oreui-active-color attribute of an element


Dark Button turns light

<div oreui-type="button" oreui-color="dark" oreui-active-color="light" onclick="OreUI.toggleActive(this)">Dark Button</div>

Light Button turns dark

<div oreui-type="button" oreui-color="light" oreui-active-color="dark" onclick="OreUI.toggleActive(this)">Light Button</div>

Red Button turns green

<div oreui-type="button" oreui-color="red" oreui-active-color="green" onclick="OreUI.toggleActive(this)">Red Button</div>

You get the point

You can also set the disabled color for buttons through oreui-disabled-color

Once disabled, you can only enable it through an outer function
as onclick becomes disabled when the button is disabled

Functions

You need to first import them


<script src="https://nspc911.github.io/OreUI-HTML/oreui.js"></script>

Active States

You can change active state of a button with three functions


OreUI.becomeActive(element)

Become Active

<div oreui-type="button" oreui-color="dark" oreui-active-color="light" onclick="OreUI.becomeActive(this)">Become Active</div>
        


OreUI.becomeInactive(element)

Become Inactive

<div oreui-type="button" oreui-color="dark" oreui-active-color="light" onclick="OreUI.becomeInactive(this)" oreui-state="active">Become Inactive</div>
        


OreUI.toggleActive(element)
Toggle Active

<div oreui-type="button" oreui-color="dark" oreui-active-color="light" onclick="OreUI.toggleActive(this)">Toggle Active</div>

Disabled States

You can change disabled state of a button with three functions


OreUI.becomeDisabled(element)
Become Disabled

<div oreui-type="button" oreui-color="dark" oreui-disabled-color="light" onclick="OreUI.becomeDisabled(this)">Become Disabled
        


OreUI.becomeEnabled(element)

You can't put this function inside a disabled div, onclick is automatically disabled in disabled elements

Enable ➡
Enable Me

<div oreui-type="button" oreui-color="dark" onclick="OreUI.becomeEnabled(document.querySelector('.enableme'))">Enable ➡</div>
<div class="enableme" oreui-type="button" oreui-color="green" oreui-disabled-color="light" disabled>Enable Me</div>
        

OreUI.toggleDisabled(element)
Toggle ➡
Toggle Me

<div oreui-type="button" oreui-color="dark" onclick="OreUI.toggleDisabled(document.querySelector('.toggleme'))">Toggle ➡</div>
<div class="toggleme" oreui-type="button" oreui-color="green" oreui-disabled-color="dark">Toggle Me</div>
